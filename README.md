# Task Manager Frontend
This folder contains the React + Vite UI for the Team Task Manager, upgraded for enterprise scale and professional collaboration.

## 🎨 Modern Workspace Features
- **Interactive Kanban Board**: A premium project detail view featuring native **Drag & Drop** task status updates.
- **Enterprise Browser**: High-performance Tasks and Projects pages with **Server-Side Pagination** and **Real-time Search**.
- **Admin Center 2.0**: Enhanced management tools for large workforces, featuring debounced user search and scrollable lists.
- **Optimistic UI**: Immediate visual feedback for drag-and-drop and task updates.
- **Rich Visuals**: Premium dark-mode aesthetics with emoji support, custom project colors, and smooth animations.
- **Real-time Sync**: Live dashboard and chat refreshes powered by Socket.IO.

## Tech Stack
- **React 19**: Modern component architecture.
- **Vite**: Ultra-fast build and dev environment.
- **Tailwind CSS**: Utility-first styling for a custom, premium look.
- **Socket.IO Client**: Real-time events and chat.
- **React Router**: Seamless navigation with protected route guards.

## Key Page Views
- `Dashboard.jsx`: Enterprise summary view with global task/project stats.
- `ProjectDetail.jsx`: The core Kanban workspace with Drag & Drop.
- `Tasks.jsx`: Power-user task browser with advanced filtering and pagination.
- `Admin.jsx`: Comprehensive company management and workforce lookup.
- `Chat.jsx`: Real-time team communication.

## Local Development

### 1. Install dependencies
```bash
cd Frontend
npm install
```

### 2. Configure Environment
Create a `.env` file:
```env
VITE_API_URL=http://localhost:5000
```

### 3. Run Development Server
```bash
npm run dev
```

## Presentation & Testing
- **Large Data Test**: Use `npm run seed` in the Backend to generate 600+ tasks, then navigate to the **Tasks** page to demonstrate smooth pagination and search.
- **Interactive Workflow**: Go to a **Project Detail** page and drag tasks between columns to show the real-time Kanban functionality.
- **Admin Search**: Open the **Admin Center** and use the search bar to find users among the hundreds generated by the seeder.
